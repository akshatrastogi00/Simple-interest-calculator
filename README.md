# Simple Interest Calculator

A lightweight web app that calculates simple interest using a principal amount, interest rate, and investment duration. Built with HTML, CSS, and JavaScript.

## Features

- Enter a principal amount
- Choose an interest rate using a slider
- Select the number of years
- See the interest outcome and projected year

## Files

- `index.html` — main user interface
- `style.css` — page styling
- `script.js` — interest calculation logic
- `extra.html` — additional file included in the project

## Usage

1. Open `index.html` in a web browser.
2. Enter the deposit amount.
3. Adjust the interest rate with the slider.
4. Select the number of years.
5. Click **Compute Interest** to view the result.

## Notes

- This app calculates simple interest using the formula `interest = principal × rate × years / 100`.
- The displayed year is based on a fixed starting year of 2020.
